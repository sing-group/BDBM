Welcome to the BDBM Help!

BDBM (BLAST DataBase Manager) provides a graphical user interface that allows the creation of high quality sequence datasets using the tools included. Using these tools, it is possible toperform several common tasks such as doing BLAST alignments, getting the open reading frames of the sequences in a Fasta file or changing the format of a Fasta file. All these task make use of several known tools such as EMBOSS, bedtools, and NCBI's BLAST, Splign & Compart, and ProSplign & ProCompart. In addition, BDBM helps keeping sequence files ordered in a file repository.

This document is mean to serve as a guide for the practical use of BDBM. It includes explanations of all of its functionalities in order to give a basic idea of usage.
