Welcome to the BDBM Help!

BDBM (BLAST Database Manager) software was developed with the goal of providing an automatic and flexible tool for detecting positively selected sites given a set of unaligned nucleotide sequence data.

An example of the usefulness of such a pipeline is given by showing, under different conditions, positively selected amino acid sites in a set of 54 Coffea putative S-RNase sequences.

This document is mean to serve as a guide for the practical use of BDBM. It includes explanations of all of its functionalities in order to give a basic idea of usage.
